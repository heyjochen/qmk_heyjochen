# Iris rev. 7 keymap

Keymap configuration files used to flash [Keebio's](https://keeb.io) Iris rev. 7 keyboard using [QMK](https://qmk.fm).

For usage information go to [QMK docs](https://docs.qmk.fm/#/newbs).

## Features

- [LAYER_TAPS](https://docs.qmk.fm/#/feature_layers?id=switching-and-toggling-layers)
- [COMBO_KEYS](https://docs.qmk.fm/#/feature_combo?id=combos)
- [CAPS_WORD](https://docs.qmk.fm/#/feature_caps_word?id=caps-word)
- [LEADER_KEY](https://docs.qmk.fm/#/reference_glossary?id=leader-key)
